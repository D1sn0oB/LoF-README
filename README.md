## 简介 / Introduction

God Reader v2 是一个功能强大的全中文电子书阅读器，专为中文用户设计。本项目完全使用中文界面，提供流畅的阅读体验。

God Reader v2 is a powerful fully Chinese e-book reader designed for Chinese users. This project features a completely Chinese interface and provides a smooth reading experience.

---

## 功能特性 / Features

### 核心功能 / Core Features

- **多格式支持** - 支持 TXT、EPUB、PDF 三种主流电子书格式
  - **Multiple Format Support** - Supports three mainstream e-book formats: TXT, EPUB, PDF

- **字体大小调整** - 根据个人喜好调整字体大小，提供最佳阅读体验
  - **Font Size Adjustment** - Adjust font size according to personal preference for the best reading experience

- **主题切换** - 支持多种阅读主题，包括浅色、深色等模式
  - **Theme Switching** - Supports multiple reading themes, including light and dark modes

- **内容滚动** - 流畅的页面滚动体验
  - **Content Scrolling** - Smooth page scrolling experience

- **批注功能** - 选择文本添加批注，方便记录阅读心得
  - **Annotation Feature** - Select text to add annotations, convenient for recording reading notes

- **翻译工具** - 内置翻译功能，支持多语言翻译
  - **Translation Tool** - Built-in translation function supporting multiple languages

- **章节导航** - 快速跳转到任意章节
  - **Chapter Navigation** - Quick jump to any chapter

## 技术栈 / Tech Stack

本项目主要使用以下技术栈：

This project mainly uses the following tech stack:

- **HTML5** - 页面结构 / Page Structure
- **CSS3** - 样式设计 / Styling
- **JavaScript** - 核心逻辑 / Core Logic

## 使用方法 / Usage

1. **导入书籍**
   - 点击左侧栏的"导入书籍"按钮，或上方工具栏的"打开文件"按钮
   - 选择您的电子书文件（TXT/EPUB/PDF）

2. **阅读功能**
   - 使用工具栏调整字体大小
   - 切换阅读主题
   - 使用滚动条浏览内容

3. **批注功能**
   - 选择文本
   - 点击"添加批注"按钮
   - 输入批注内容

4. **翻译功能**
   - 选择需要翻译的文本
   - 使用内置翻译工具进行翻译

1. **Import Books**
   - Click the "Import Books" button on the left sidebar, or the "Open File" button on the top toolbar
   - Select your e-book file (TXT/EPUB/PDF)

2. **Reading Features**
   - Use the toolbar to adjust font size
   - Switch reading themes
   - Browse content using the scroll bar

3. **Annotation Feature**
   - Select text
   - Click the "Add Annotation" button
   - Enter annotation content

4. **Translation Feature**
   - Select the text to be translated
   - Use the built-in translation tool for translation
